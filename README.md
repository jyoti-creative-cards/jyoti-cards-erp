# Jyoti Cards ERP

Internal stock, purchase, sales, inventory, and payments system built with Streamlit and SQLite.

## Run

```bash
pip3 install -r requirements.txt
streamlit run app.py
```

## Features

- Product, vendor, customer masters
- Product image upload
- Purchase orders add stock
- Sales orders reduce stock
- Inventory transaction log
- Customer and vendor payments
- Ledger views

## Smoke test

```bash
python3 smoke_test.py
```
